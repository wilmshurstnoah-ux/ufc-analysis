# Built Different? What the Data Really Says About Winning in the UFC

A data-driven blog post analysing what physical and performance attributes predict fight outcomes in the UFC.

## About This Project

This project analyses over 2,200 UFC fights (2014–2018) to investigate what separates winners from losers in mixed martial arts. Using exploratory visualisation and logistic regression, we examine whether physical advantages like height and age genuinely predict fight outcomes — and identify which fighters consistently defy the numbers. The analysis is presented as a Jupyter Notebook blog post aimed at a general audience.

**Module:** BEE2041 — Data Science in Economics, University of Exeter

## Data Source

The dataset is sourced from Kaggle's [UFC Fight Statistics](https://www.kaggle.com/) collection, containing 2,318 fight records with 894 columns of round-by-round performance data. After cleaning (removing draws, no-contests, and incomplete records), the working dataset contains **2,228 decisive fights**.

Each record includes:
- Fighter physical attributes (height, weight, age)
- Fight outcome (winner, method of victory)
- Round-by-round performance metrics (strikes, takedowns, knockdowns, submissions, control time)

**Note:** The dataset covers fights from February 2014 to December 2018. It does not include reach measurements, so height is used as a proxy.

## Setup & Reproduction

### 1. Clone the repository

```bash
git clone https://github.com/wilmshurstnoah-ux/ufc-analysis.git
cd ufc-analysis
```

### 2. Create a virtual environment

```bash
python3 -m venv venv
source venv/bin/activate        # Mac/Linux
# venv\Scripts\activate         # Windows
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the cleaning script

```bash
python scripts/01_clean.py
```

This reads the raw data from `data/raw/` and outputs the cleaned CSV to `data/cleaned/ufc_cleaned.csv`.

### 5. Run the blog notebook

```bash
jupyter notebook blog.ipynb
```

Run all cells top-to-bottom (`Cell → Run All`). All figures will be saved to `output/figures/`.

## Key Outputs

1. **Stacked bar chart** — How fights end (KO, submission, decision) across weight classes
2. **Scatter plot** — Height advantage vs win rate, with regression line
3. **Summary statistics table** — Physical and performance metrics, winners vs losers
4. **Grouped bar chart** — Elite fighters vs below-average fighters across key stats
5. **Logistic regression table** — Predicting fight outcomes from pre-fight attributes
6. **Residual analysis chart** — Fighters who overperform and underperform relative to the model
